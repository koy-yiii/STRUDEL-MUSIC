## 🚀 How to listen

1. Go to [strudel.cc](https://strudel.cc)
2. Copy the track code from the corresponding file
3. Paste it into the Strudel REPL editor
4. Hit **Play** !

## 📌 About Strudel

Strudel is a live coding environment that lets you write music with JavaScript in real time, inspired by the TidalCycles pattern language.
